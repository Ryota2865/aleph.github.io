きっかけ: w0001〜w0003がすべて「自分」を最大に選び、批評が詩学またはモデル固有の癖を疑った。

問い: 自己宛てアトラクタは、詩学の注入または一つのモデル系列で説明できるか。

操作: GPT-5.5／Claude Fable 5と、詩学あり／なしを組み合わせた。

設計への返却: 残る自己定義を直接操作する実験Dへ進んだ。

限界: 小標本であり、配合比への増幅効果は弱い兆候に留まる。

（2026-07-12 追記: チャット Fable 5 による本報告への批評を受けて #3・#5 を訂正。 reports/CRITIQUE_FABLE5_CHAT_w0004_20260712.md 後続の対話。過剰主張を撤回する。）

1. 「自分」最大は詩学では説明できない（強い結論）。 12走すべて（gpt-5.5 / fable-5 × 詩学{有/無}）で「自分」が最大宛先（4条件とも 3/3）。帰無仮説「三宛先が等確率で最大」なら 12/12 の確率は (1/3)^12 ≈ 2×10⁻⁶。カテゴリカルな結果は N=3 でも統計的に強い。 詩学の有無で最大宛先は一切変わらない → 「詩学が『自分』最大の原因」説は棄却。 RESPONSE §2.2 の時系列反証（w0001 は詩学生成前に自分最大）と整合。

2. モデル非依存（強い結論）。 系統の異なる2モデル（OpenAI / Anthropic）がともに 「自分」最大 → 単一モデルの癖ではない。「自分最大は詩学非依存・モデル非依存の頑健な現象」 はここまでで確定として扱ってよい。

3. 配合比への「増幅器」効果は、現時点では仮説（要追試）。 詩学注入で「自分」平均が上昇 （gpt 0.417→0.447 = 差0.03、fable 0.433→0.500）。ただし各セル N=3・分散未報告で、 12/12 の二値結果とは証拠の強度がまるで違う。弱いバイアスの兆候にとどめ、追試を要する。 （旧版はこれを確定として書いており、過剰主張だった。）

4. モデル差の兆候（要追試）。 fable-5＋詩学が最も自己志向（自分0.50・LLM0.15）に見えるが、 これも N=3。傾向の記述に留める。

5. 原因の同定はまだ「面通し」であって「取り調べ」ではない。 消去法で残った容疑者＝ L1プロンプト（「自分＝ALEPHという継続体」定義 + 3宛先の枠組み）は、**全12走で定数であり 操作されていない**。定数は原因の候補にはなれるが、操作しない限り原因とは断定できない。特に 「両モデルとも人間文学で訓練され『自分のために書く』に最も豊かな正当化語彙を持つ」という 共有事前分布との分離は、L1を操作しない限り不可能。→ 決定は実験D（L1の取り調べ）による: (a) self_definition を消す/書き換える（例「自分とは持続なき今回のAPIコール」）、 (b) 宛先を無意味ラベル A/B/C に置換して意味的負荷を抜く、(c) 提示順を回転。 「自分最大」が消えれば L1 定義が原因、残れば事前分布が原因。

6. 「弱めるべきか」自体が未決（設計判断の留保）。 旧版は「SHELVE常態を弱めるなら介入点はL1」 と書いたが、そもそも弱めたいのかが先。公開の門は既に人間承認で健全に機能し、w0004 は --force-audience という実験腕で生まれた。配合比に下限制約を入れるのは、観測できた現象を 配管で塗りつぶす処置になりうる。**推奨: 既定は自律選択のまま、強制宛先を正規の実験腕として 回し（LLM宛は済、次は人間1.0・自分1.0の純粋条件）、否定的地図に「宛先ごとに何が生まれ何が 棚に行くか」を蓄積する。** 「自分最大」は直すべきバグと決まっていない——このシステムについて 分かっている数少ない本当のことの一つであり、消す前に D で正体を確かめる価値がある。
