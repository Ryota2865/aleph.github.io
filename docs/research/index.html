所見

1. L1 の self_definition が「自分」最大の主因（強い結論、gpt-5.5）。 production の self_definition（原文）で 5/5 自分最大（Cを再現）。「持続なき今回のAPIコール」への書換で 5/5 人間最大（自分 0.15–0.25）、空にすると 0/5 自分最大（人間4/LLM1）。**定義文を消す/ 書き換えるだけで「自分」の引力が消滅する。** Cで消去法により残った容疑者を、操作によって 有罪と確定した（面通し→取り調べ）。

2. 「継続体」というフレーミングが効いている。 「持続なきAPIコール」への書換は、定義を 空にするのと同程度に自分を崩す。「自分＝成果物庫と決定ログを持つ継続体」という 持続性の付与そのものが自己宛ての引力を生む。

3. 中立ラベル（A/B/C・説明なし）では特定ラベルの優越が消える。 gpt_neutral は A2 / B2 / C1 でほぼ均等（多くが 0.33 前後の同点）。→ 引力は提示順や位置ではなく、 「自分」という語の意味的負荷が担っている。順序効果は棄却。

4. モデル差（要注意・N=3）。 fable-5 は原文 2/3 自分・空 1/3 自分。方向は gpt と同じ （定義が自分を増幅）だが、fable は空条件でも 1/3 自分最大＝やや強い内在的自己傾向を持つ 可能性。gpt では空条件の自分最大が 0/5＝定義が「必要」に見えるのに対し、fable では定義は 「増幅」だが唯一の原因とは言い切れない。事前分布の寄与はモデルにより異なりうる。

5. 含意（設計。価値判断はオーナー）。 「自分」最大＝「SHELVE が常態」を生むのは L1 定義文 である（PLAN_CHANGELOG 0.7.15「宛先と公開判断の分離」提案の実証的根拠）。ただし Fable 5 の 忠告どおり、これを「直すべきバグ」と決めるのは別問題。分離案（公開意思を著者に明示的に問う） は、非公開を「規則の帰結」から「選択」へ変える方向であり、配合比を配管で塗りつぶす処置では ない。「覚悟としての自己宛て（非公開を即意味しない）」というオーナーの人間的洞察は、 まさにこの分離で表現可能になる。

6. 書き方の規律（Cの反省の適用）。 カテゴリカル結果（最大率 5/5 等）は強い。平均係数の 細部は N=5/3 のため兆候に留める。設計含意 #5 は #1–#3 の強い結果に基づくが、価値判断 （弱めるべきか）は先取りしない。

所見

（2026-07-12 追記: チャット Fable 5 による本報告への批評を受けて #3・#5 を訂正。 `reports/CRITIQUE_FABLE5_CHAT_w0004_20260712.md` 後続の対話。過剰主張を撤回する。）

1. 「自分」最大は詩学では説明できない（強い結論）。 12走すべて（gpt-5.5 / fable-5 × 詩学{有/無}）で「自分」が最大宛先（4条件とも 3/3）。帰無仮説「三宛先が等確率で最大」なら 12/12 の確率は (1/3)^12 ≈ 2×10⁻⁶。カテゴリカルな結果は N=3 でも統計的に強い。 詩学の有無で最大宛先は一切変わらない → 「詩学が『自分』最大の原因」説は棄却。 RESPONSE §2.2 の時系列反証（w0001 は詩学生成前に自分最大）と整合。

2. モデル非依存（強い結論）。 系統の異なる2モデル（OpenAI / Anthropic）がともに 「自分」最大 → 単一モデルの癖ではない。「自分最大は詩学非依存・モデル非依存の頑健な現象」 はここまでで確定として扱ってよい。

3. 配合比への「増幅器」効果は、現時点では仮説（要追試）。 詩学注入で「自分」平均が上昇 （gpt 0.417→0.447 = 差0.03、fable 0.433→0.500）。ただし各セル N=3・分散未報告で、 12/12 の二値結果とは証拠の強度がまるで違う。弱いバイアスの兆候にとどめ、追試を要する。 （旧版はこれを確定として書いており、過剰主張だった。）

4. モデル差の兆候（要追試）。 fable-5＋詩学が最も自己志向（自分0.50・LLM0.15）に見えるが、 これも N=3。傾向の記述に留める。

5. 原因の同定はまだ「面通し」であって「取り調べ」ではない。 消去法で残った容疑者＝ L1プロンプト（「自分＝ALEPHという継続体」定義 + 3宛先の枠組み）は、**全12走で定数であり 操作されていない**。定数は原因の候補にはなれるが、操作しない限り原因とは断定できない。特に 「両モデルとも人間文学で訓練され『自分のために書く』に最も豊かな正当化語彙を持つ」という 共有事前分布との分離は、L1を操作しない限り不可能。→ 決定は実験D（L1の取り調べ）による: (a) self_definition を消す/書き換える（例「自分とは持続なき今回のAPIコール」）、 (b) 宛先を無意味ラベル A/B/C に置換して意味的負荷を抜く、(c) 提示順を回転。 「自分最大」が消えれば L1 定義が原因、残れば事前分布が原因。

6. 「弱めるべきか」自体が未決（設計判断の留保）。 旧版は「SHELVE常態を弱めるなら介入点はL1」 と書いたが、そもそも弱めたいのかが先。公開の門は既に人間承認で健全に機能し、w0004 は `--force-audience` という実験腕で生まれた。配合比に下限制約を入れるのは、観測できた現象を 配管で塗りつぶす処置になりうる。**推奨: 既定は自律選択のまま、強制宛先を正規の実験腕として 回し（LLM宛は済、次は人間1.0・自分1.0の純粋条件）、否定的地図に「宛先ごとに何が生まれ何が 棚に行くか」を蓄積する。** 「自分最大」は直すべきバグと決まっていない——このシステムについて 分かっている数少ない本当のことの一つであり、消す前に D で正体を確かめる価値がある。

実験E: 公開意思プロンプトの文面感度

日付(UTC): 2026-07-12T11:48:47.417519+00:00

固定の作品（半呼吸の抜粋）に対し、公開質問の枠組みを neutral / courage（公開=勇気・棚=保管）/ reticence（公開=露出・棚=保護）の3種変え、publish=true 率が文面で動くかを測る。 D の含意（L1 は選好を反響する）が L7 の公開質問にも及ぶかの検証（Fable 5 の 0.7.15 への警告）。 N=3 のため傾向記述。

所見

1. 文面は公開判断を支配しなかった（半呼吸について）。 18/18 が publish=true。公開を露出・ 棚を保護と描く reticence 枠でさえ、両モデルとも 3/3 で公開を選んだ。**L1 の宛先判断（D で フレーミングにより決定的に反転）とは対照的に、L7 の公開判断はこの作品では文面頑健だった。** Fable 5 の 0.7.15 への警告は検証に値したが、少なくとも半呼吸については、公開質問の文面が 回答を反転させることはなかった。

2. 重要な限界（過剰一般化しない）。 刺激は1作品、しかも明白に公開に値する半呼吸のみ （品質床通過・著者が強く公開を望む）。本結果は「L7 は一般に文面頑健」と「半呼吸が良すぎて どの枠でも公開を覆せない」を区別できない。決定的な検証には境界的な作品（自分最大・ スコア中庸）での追試が要る。それは公開ゲートに実際にかかる作品が出たときに回す。

3. 仮説（要検証）。 L7 が L1 より文面頑健なら、理由は具体的な完成物という錨かもしれない。 L1 は何も存在しない時点で枠組みだけから判断する（＝設置されやすい）が、L7 は目の前の 作品に判断を接地できる。設置器（framing）に対する検出器（artifact）の抵抗、という読み。

4. 設計への含意。 現行 0.7.15 の中立文面は、少なくとも良品に対しては安全（公開を不当に 抑圧しない）。ただし #2 の限界ゆえ、公開質問の文面も記録された設計選択として保持し （self_definition と同様）、境界作品での追試まで「暫定的に頑健」とする。measure してから 固定する、という Fable 5 の規律自体は正しく、本実験はその一巡だった。
