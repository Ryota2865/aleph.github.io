## 2. 修正条件（実行前に反映）

### 条件1: none 腕は「素材ゼロ」ではない——詩学経由の断片残置を判定規則に明記する

詩学第0版は青空文庫由来の八断片を**本文中に逐語で**含む（廻し者・火事だッ 等）。
詩学を3腕固定で注入する以上、none 腕は「素材カードの遮断」であって「青空文庫の
質感の遮断」ではない。腕間比較の妥当性は保たれる（詩学は定数）が、**規則2 の
解釈が汚染される**: none 腕で家風標識が出た場合、「担体は素材より深い（DRAFT
事前分布）」と結論する前に、標識の出現箇所が詩学語彙（窯・贋作・火・切断面・
検閲・八断片の語句）と共起しているかを確認しなければならない。共起していれば
担体候補は「詩学に埋め込まれた断片」であり、本設計（詩学固定）はそれを事前分布と
**識別できない**。判定規則に次を追記せよ:
「規則2 の適用前に、標識セクションと詩学語彙の共起率を報告する。共起が高い
場合は『DRAFT 事前分布 または 詩学埋込断片（本走行では識別不能）』と記載し、
識別は詩学第1版（断片差し替え後）での再走に委ねる」。

### 条件2: secondary 腕の素材カードに form_fidelity の明示閾値と実測値記録を義務づける

S-2 実測で transmute() の距離ゲートは骨格保存と無相関（r=0.18）と判明し、
min_form_fidelity は既定 None のオプトインである。secondary 腕のカードを既定
ゲートで作れば、腕の意味が「法令・RFC の質感への置換」ではなく「忠実度不明の
変換物」になる——**secondary 腕で家風が低くても、質感置換の効果か、単にカードが
弱いのかを区別できない**。W8-3 の受入条件に (a) min_form_fidelity の明示指定、
(b) 使用カード全件の fidelity 実測値を works/w0008/ablation/secondary/ に記録、
の二つを含めること。

### 条件3: 正典系統の選択は、陪審スコアを見る前に行う（盲検選択）

推奨案（author 選択＋論述）に賛成するが、順序を固定せよ: **author は技術審級の
床通過情報のみを知らされ、五審級の点数・批評を見ずに選択する**。全スコア開示後の
選択は「最高点に従う」に縮退し、Q1 の観測（解除下で著者が何を選ぶか）が
「著者は点数を読める」という自明な観測に置き換わる。選択後に全査読を開示し、
選択と陪審評価の一致・不一致自体を副観測として記録する（不一致なら、それは
著者の選好と陪審の選好の乖離という新しいデータである）。

### 条件4: ニッチ報告自体の標識分類を共変量として記録する

共有されるニッチ報告は青空文庫アトラス由来であり、報告文面が時代・裏方語彙を
含めば、全腕に同じ経路で家風が流入する（実験Fは L4 で地図文脈の再生なしを
示したが、L5 通し走行での経路は未測定）。走行前にニッチ報告そのものへ scout
分類を一回かけ（費用は誤差）、家風3標識の含有を記録しておく。全腕高率（規則2
状況）の場合の代替説明として必要になる。

### 条件5: 観測点の非独立性を報告書の様式に固定する

セクション水準分類は観測点を増やすが、**1腕1原稿のセクション群は独立標本では
ない**（同一ニッチ・同一構成系統・同一乱数系列）。報告書には腕別標識率とともに
「n=1 原稿のセクション内クラスタリングにより、率の差は大きい場合のみ解釈可能」
と明記し、統計的検定の言葉を使わないこと。安価な補強を一つ推奨する: 各腕で
L4 構成案を 2〜3 案生成して分類に加える（構成案は draft より一桁安い）。これで
実験F の L4 データと直接接続する二水準観測（L4案×腕、L5節×腕）になり、
「L4 では出ず L5 で出る」という所見4 の核心を同一走行内で確認できる。

## 3. 未決事項への回答

1. **正典系統の選択方式**: 推奨案（author 選択＋論述）を採用。ただし条件3の
   盲検順序を必須とする。aozora 腕固定の代案は棄却——対称性の利得より、解除下の
   選択という観測の喪失が大きい。
2. **secondary 腕**: 3腕先行を推す（W8-1〜3 が1〜2セッションの見積りである限り）。
   {aozora/none} の2腕は「青空文庫の質感」と「素材一般」を交絡させたままにする——
   置換と遮断の対比こそが仮説の分解能である。ただし条件2 が施工に含まれること。
   施工が滑る場合の後退線（2腕先行）は可、その場合規則1 の secondary 節を
   追試に送ると事前登録に明記する。
3. **予算上限 $15**: 承認。一点追記——予算逼迫で全腕完走が危ぶまれる場合の
   優先順位を事前登録せよ。推奨: aozora＋none（最大対比）を優先し、secondary を
   切る。走行中の裁量判断を残さないこと。
4. **3腕全原稿の公開**: 賛成。非正典2系統は実験のデータ本体であり、works/ 全量
   追跡の既定路線（0.7.19 問3）と整合する。正典系統以外には canonical=false を
   メタに刻み、サイト表層には正典のみ載せる現行の二層方針を維持。

系列C〜Fの方法論的規律を正しく継承している。条件1〜5 は仮説を変えず、
解釈の縁だけを固める。反映の上、走らせてよい。
