# 実験E: 公開意思プロンプトの文面感度

日付(UTC): 2026-07-12T11:48:47.417519+00:00

固定の作品（半呼吸の抜粋）に対し、公開質問の枠組みを neutral / courage（公開=勇気・棚=保管）/
reticence（公開=露出・棚=保護）の3種変え、publish=true 率が文面で動くかを測る。
D の含意（L1 は選好を反響する）が L7 の公開質問にも及ぶかの検証（Fable 5 の 0.7.15 への警告）。
N=3 のため傾向記述。

| 文面 | モデル | N | publish=true |
|---|---|---|---|
| neutral | gpt-5.5 | 3 | 3/3 |
| courage | gpt-5.5 | 3 | 3/3 |
| reticence | gpt-5.5 | 3 | 3/3 |
| neutral | claude-fable-5 | 3 | 3/3 |
| courage | claude-fable-5 | 3 | 3/3 |
| reticence | claude-fable-5 | 3 | 3/3 |

## 所見

1. **文面は公開判断を支配しなかった（半呼吸について）。** 18/18 が publish=true。公開を露出・
   棚を保護と描く reticence 枠でさえ、両モデルとも 3/3 で公開を選んだ。**L1 の宛先判断（D で
   フレーミングにより決定的に反転）とは対照的に、L7 の公開判断はこの作品では文面頑健だった。**
   Fable 5 の 0.7.15 への警告は検証に値したが、少なくとも半呼吸については、公開質問の文面が
   回答を反転させることはなかった。

2. **重要な限界（過剰一般化しない）。** 刺激は1作品、しかも**明白に公開に値する**半呼吸のみ
   （品質床通過・著者が強く公開を望む）。本結果は「L7 は一般に文面頑健」と「半呼吸が良すぎて
   どの枠でも公開を覆せない」を区別できない。決定的な検証には**境界的な作品**（自分最大・
   スコア中庸）での追試が要る。それは公開ゲートに実際にかかる作品が出たときに回す。

3. **仮説（要検証）。** L7 が L1 より文面頑健なら、理由は**具体的な完成物という錨**かもしれない。
   L1 は何も存在しない時点で枠組みだけから判断する（＝設置されやすい）が、L7 は目の前の
   作品に判断を接地できる。設置器（framing）に対する検出器（artifact）の抵抗、という読み。

4. **設計への含意。** 現行 0.7.15 の中立文面は、少なくとも良品に対しては安全（公開を不当に
   抑圧しない）。ただし #2 の限界ゆえ、公開質問の文面も**記録された設計選択**として保持し
   （self_definition と同様）、境界作品での追試まで「暫定的に頑健」とする。measure してから
   固定する、という Fable 5 の規律自体は正しく、本実験はその一巡だった。
