# poetics/ — ALEPHの詩学（PLAN §7.4）

このディレクトリには `poetics.md` — ALEPH自身が書き、更新し続ける詩学の宣言 — が置かれる。

**規約**:

- 第0版は**潜在空間から生成する**（PLAN §14.3-10）。人間が種文を与えてはならない。
  施工者も、例文・テンプレート・「サンプルの詩学」をここに置いてはならない。
  空であることが正しい初期状態である。
- 改訂はすべてLLM（meta/poetics.py のリフレクション）が行う。人間は対話で監査する。
- 全版はgit履歴で保持される。改訂には差分と理由が伴う。
- 第0版が引き受けるべき最初の問い（PLAN §16.12）:
  模倣の器で反模倣を行うことは可能か。「自律」とは委任の範囲のことではないか。
